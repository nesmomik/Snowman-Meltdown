# Snowman-Meltdown

A game where the player guesses letters to reveal a secret word. With each incorrect guess, the snowman (drawn with ASCII art) slowly “melts.”

Install by executing in a terminal 
'''
git clone https://github.com/nesmomik/Snowman-Meltdown.git
''' 

Change into the directory with 
'''
cd Snowman-Meltdown
'''

Start the game with 
'''
python snowman.py
'''
