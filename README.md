# Snowman-Meltdown

A game where the player guesses letters to reveal a secret word. With each incorrect guess, the snowman (drawn with ASCII art) slowly “melts.”

Install by executing '''git clone https://github.com/nesmomik/Snowman-Meltdown.git''' in a terminal.

Change into the directory with '''cd Snowman-Meltdown'''.

Start the game with '''python snowman.py'''.
